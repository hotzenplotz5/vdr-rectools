## Status

VDR-Rectools befindet sich im Wartungsmodus.

Neue Funktionen werden nur ergänzt, wenn sie unmittelbar für die Aufnahmeverwaltung benötigt werden.

Die Hauptentwicklung erfolgt zukünftig im Projekt VDR-Suite.
